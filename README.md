
## UtilsVerse


## Installation
```js
npm i utilsverse
```
 [npm Package.](https://www.npmjs.com/package/utilsverse)



## Example
```js
// import module
import {debounce} from 'utilsverse/camelCase';

// debounce function
debounce(()=>{
  // callback
},time);

Ex - 
debounce(()=>{
  // callback
},1000);
```
