
## UtilsVerse


## Installation
```js
npm i utilsverse
```

## Example
```js
// import module
import {debounce} from 'utilsverse/camelCase';

// debounce function
debounce(()=>{
  // callback
},time);

Ex - 
debounce(()=>{
  // callback
},1000);
```
